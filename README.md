#  aw-watcher-project

## Overview

System tray app for ActivityWatch to report current project

## Getting Started

Install `aw-watcher-project`:

```
pip install aw-watcher-project
```

A simple example:

```python
import aw_watcher_project

# Do something with aw_watcher_project
```
